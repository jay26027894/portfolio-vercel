# Cosmic Explorer - 3D Portfolio

A 3D immersive portfolio website built with **React**, **Three.js**, and **Tailwind CSS**. This project showcases an interactive journey through a "digital universe," featuring toon-shaded 3D elements, smooth scroll animations, and a unique storytelling approach.

*   **Immersive 3D Experience:** Navigate through a 3D space with scrolling controls.
*   **Toon Shading:** Custom toon shader implementation for a stylized look.
*   **Interactive Elements:** Floating debris, orbiting rings, and hovering crystals.
*   **Storytelling Acts:**
    *   **Act I: The Origin:** Hero section introducing the developer.
    *   **Act II: The Artifacts:** Visual representation of technical skills.
    *   **Act III: The Journey:** Experience timeline represented as pillars.
    *   **Act IV: Constellations:** Project showcase with geometric shapes.
    *   **Act V: New Horizons:** Contact section.
*   **Responsive Design:** Optimized for different screen sizes.
